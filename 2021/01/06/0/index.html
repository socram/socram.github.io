Titulo bem grande para a pagina

06 January, 2021

Trio is a simple, content-focused, responsive, mobile-friendly Jekyll theme. Trio comes with support for these things out-of-the-box:

  • RSS feed
  • 404 page
  • Markdown syntax highlighting, fenced code blocks, tables, GitHub gists, blockquotes
  • Post sharing buttons
  • Paginated list of posts
  • Formspree contact form
  • Nine color themes
  • Support for logos using Font Awesome icons
  • Full-text search


Where is it?

Trio lives in a GitHub repository. It is free and open-source with MIT License.

Please feel free to download it, request a feature or report a bug.
